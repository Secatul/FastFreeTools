# 🧰 Fast Free Tools

🚀 Uma coleção de ferramentas rápidas, gratuitas e úteis para tarefas do dia a dia, como conversão de dados, análise de texto, utilitários e muito mais.

---

## ✨ Funcionalidades

- 🔐 Codificador/Decodificador Base64  
- ✍️ Contador de palavras e caracteres  
- 📊 Gerador de tabelas de dados  
- 🌐 Analisador de User Agent  
- 🔍 Analisador de meta tags SEO  
- 🔢 Contadores e utilitários diversos  
- 📱 Interface responsiva e moderna  

---

## 🧠 Estrutura do Projeto

### 🏠 Página Inicial
- Exibe todas as ferramentas disponíveis  
- Campo de busca para encontrar ferramentas rapidamente  
- Filtros por categoria (Programação, Texto, Segurança, etc.)  

### 🧩 Ferramentas
Cada ferramenta possui sua própria interface com:

- 📥 Entrada de dados  
- ⚙️ Processamento em tempo real  
- 📤 Resultado instantâneo  
- 🧼 Interface simples e intuitiva

## 🌍 Internacionalização

O projeto conta com suporte a múltiplos idiomas utilizando **next-intl**, permitindo:

- 🌐 Rotas internacionalizadas (`/en`, `/pt-br`, `/es`, `/fr`, `/de` etc.)
- 🔄 Troca de idioma dinâmica
- 📂 Estrutura organizada de traduções
- 🧩 Suporte completo em todas as ferramentas

```bash

# Acesse a pasta
cd FastFreeTools

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
